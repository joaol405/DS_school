A linguagem de Marcação de Hipertexto (html) é uma linguagem de computador que compõe a maior parte da páginas da internet e dos aplicativos online. Um Hipertexto é um texto usado para fazer referêcia a outros textos, enquanto uma linguagem de marcação é composta por uma série de marcações que dizem para os servidores da web qual é o estilo e a estrutura de um documento

é facil de aprender veja a seguir:

Texto Responsivo em um Contêiner

EXEMPLO:

/* Estilizando um elemento de parágrafo (p) */ p { font-family: Arial, sans-serif; /* Define a família de fontes */ font-size: 16px; /* Define o tamanho da fonte */ color: #333; /* Define a cor do texto */ margin-bottom: 20px; /* Define a margem inferior */ line-height: 1.5; /* Define a altura da linha */ }
